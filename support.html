How It Works

  1. Create a group: Start by setting up a new group and inviting members.
  2. Propose activities: Members can propose activities and define potential timeframes.
  3. Vote on preferences: Invitees vote on their preferred times using a thumbs up/down system.
  4. AI scheduling: Our AI evaluates all inputs and finds the best time slot for the event.
  5. Event confirmation: Once finalized, the event details are shared with the group.
